HopeHub - Mobile Application for Drug Addiction Recovery in Sri Lanka

Project Overview

HopeHub is a multilingual AI-driven mobile application designed to support drug addiction recovery in Sri Lanka through personalized assessment, counselling, relapse monitoring, and life rebuilding support.

The system combines AI technologies, behavioral analysis, and recovery assistance to help users manage addiction recovery more effectively and safely.

🛠️ System Setup

1 .Clone Repository
git clone <your-github-repository-link>


2 .Frontend Setup
cd FE
npm install


## Run Frontend
npx expo start


3 .Backend Setup
cd backend
npm install

4 .Run Backend
npm run dev

5 .Deploy Backend using ngrok
ngrok http 5000

Copy the generated ngrok URL and use it inside the frontend API configuration.
